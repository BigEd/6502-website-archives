  • Syntax table chart

The columns of the syntax table chart are:

  1. Category number
  2. Address (within the syntax table)
  3. Verb token number (used as an index to the precedence and jump tables at $E980, $EA00, and $EA80)
  4. Precedence
  5. Address of verb's action (or $FFFF)
  6. Characters in the syntax table (if any)
  7. Syntax token(s)

Syntax tokens:

  • 00 return with syntax error
  • 01 parse string literal character (up to closing quote or end of line)
  • 02 parse REM character (up to end of line)
  • 03 return sucessfully (no error)
  • 00+category required
  • 20+category optional
  • 40+category end of category
  • 60+category end of rule
  • 7E branch "backward" 1 byte
  • 7F branch "backward" 2 bytes

sp (space) matches a letter A-Z, ! matches a digit 0-9, and _ matches the end of the line.
