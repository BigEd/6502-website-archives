Installation instructions

  1. Start with the bootstrap code
  2. Use the bootstrap code to install the "hex" file code
  3. Use the "hex" file code to install the relocator
  4. Use the "hex" file code to install the assembler
  5. If necessary, use the relocator to adapt the assembler for your 6502-based system.
